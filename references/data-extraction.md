# 数据提取与 Excel 契约

## 1. 制定指标清单

先抽读 2–3 份 TXT 的目录、财务摘要和报表结构；每个市场至少一份。根据实际披露选择指标，覆盖：

- 损益、资产负债、现金流核心科目
- 盈利能力、偿债能力、运营效率和现金质量
- 分部、管理层讨论、经营数据及行业特有指标

只保留能支持趋势判断、同行对比或风险识别的数据点。记录未披露的分析盲区。

## 2. 建立 TSV 模板

为每家公司 × 每期间创建 `{output_dir}/tsv/{公司}_{期间}.tsv`：

```text
指标名称\t数据\t备注
营业收入\t-\t-
归母净利润\t-\t-
```

第一列使用统一指标名且不得被提取过程修改。数据统一为亿元或百分比；备注记录章节/页码、原始科目名、原始值和单位。

## 3. 提取验收

标记缺失前至少满足以下两项：

1. 使用至少三个关键词变体（含简繁体或英文）全文搜索。
2. 从目录定位到应披露章节并检查。
3. 检查财务摘要、管理层讨论与分析、完整报表和相关附注。
4. 确认指标对该公司不适用并说明原因。

以下指标至少用两个独立披露位置交叉核对：营业收入、毛利、归母净利润、总资产、归母权益、有息负债/总借贷、经营活动现金流、行业核心经营指标。

如果使用子 Agent，仅让其编辑预置 TSV；按同一市场 2–3 家公司分组。要求其只返回提取数量、缺失举证、单位换算问题和口径差异，不返回 TXT 或 TSV 正文。

每个批次返回前报告：

- 指标总数、已提取数、缺失数
- 每个缺失项的关键词和已检查章节
- 单位换算的原始值与原始单位
- 不同公司/期间的科目名称和口径差异

## 4. 统一口径与复核

统一科目映射和单位。复核：

- 资产 = 负债 + 权益等勾稽关系
- 毛利、毛利率及其他可重算指标
- 多期间同比变化和异常数量级
- 同一公司不同披露位置的差异
- 公司间会计口径、合并范围和期间差异

无法消除的差异不要强行合并，在备注和报告素材中明确披露。

## 5. 生成 Excel

创建 `{output_dir}/excel_config.json`：

```json
{
  "companies": [
    {
      "name": "公司A",
      "periods": {
        "2024": "{output_dir}/tsv/公司A_2024.tsv",
        "2025": "{output_dir}/tsv/公司A_2025.tsv"
      }
    }
  ],
  "indicators": ["营业收入", "归母净利润", "总资产", "资产负债率", "平均售价"],
  "indicator_definitions": {
    "平均售价": {
      "canonical_unit": "万元/平方米",
      "accepted_units": {
        "万元/平方米": 1,
        "元/平方米": 0.0001
      }
    }
  },
  "output": "{output_dir}/数据底稿.xlsx"
}
```

`generate_excel.py` 内置常见金额、面积、比率、平均售价和每股盈利的标准单位规则；`indicator_definitions` 用于覆盖或补充项目特有指标。每个规则必须声明标准单位和允许的原始单位换算系数。出现未知单位、同一未定义指标混用多个单位、重复指标或无效 TSV 表头时必须停止生成，不能静默写入 Excel。

执行：

```bash
python {SD}/generate_excel.py --config "{output_dir}/excel_config.json"
```

第一个 Sheet 是指标 × 公司-期间汇总；后续 Sheet 保留指标名称、数据和来源备注。不要为了排版把分析数据拆成更多 Sheet。

汇总 Sheet 使用换算后的标准单位纯数字；明细 Sheet 保留 TSV 原始值和备注，不覆盖原始披露口径。

## 6. 汇总表格式

在对话和 `报告素材.md` 中使用单位单列、数字纯值的透视表。公司按用户指定顺序，每家公司内期间升序；先绝对值，再比率。不要在汇总表添加来源长文本。
